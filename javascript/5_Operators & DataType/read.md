There are different types of JavaScript operators:

* Arithmetic Operators

   +	Addition
   -	Subtraction
   *	Multiplication
   **	Exponentiation (ES2016)
   /	Division (quotient )
   %  	Modulus (Remainder)
   ++ 	Increment
   -- 	Decrement


* Assignment Operators

=	   x = y   	x = y
+= 	x += y	x = x + y
-= 	x -= y	x = x - y
*= 	x *= y	x = x * y
/= 	x /= y	x = x / y
%= 	x %= y	x = x % y
**=   x **= y	x = x ** y


* Comparison Operators
* Logical Operators
* Ternary Operators
* String Operators
* Bitwise Operators
* Type Operators
